# Deutsche Plotsquard Übersetzung
In dieser Repo befindet sich eine **Deutsche Plotsquared** Config!

Diese enthält zwar noch einige Gramatikfehler (Sorry), aber werde ich nach und nach ausbauen!

# Wie installiere ich die Language?
1. Um die Language zu installieren lädst du dir zuersts den neusten Release herunter!
2. Nun gehst du in deinem **/plugins Ordner**!
3. In deinem Plugins Ordner wechselst du zum Verzeichnis **/plugins/PlotSquared/lang/messages_en.json**
4. Lösche diese, und lade die gerade heruntergeladene .yml Datei hoch!
5. Lade dein PlotSquared nun neu über /plot reload!
6. Viel Spaß :)

Ich freue mich über jede Korrektur im Issues/Pull Requests Tab!
